# Benchmarks

## Overview

QONTOS benchmark suites, results, and analysis tooling are maintained in the dedicated benchmarks repository.

## Repository

See [qontos-benchmarks](https://github.com/qontos/qontos-benchmarks) for:

- Benchmark circuit definitions and test suites
- Hardware performance metrics and historical results
- Comparison frameworks for cross-platform evaluation
- Reproducibility tooling and data formats

## Benchmark Categories

| Category | Description |
|----------|-------------|
| Gate fidelity | Single- and two-qubit gate error rates |
| Circuit volume | Quantum volume and related volumetric benchmarks |
| Error correction | Logical error rates vs. code distance |
| Application | End-to-end performance on targeted algorithms |
| System | Throughput, latency, and uptime metrics |
