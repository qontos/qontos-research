# Security Policy

## Reporting a Vulnerability

This repository contains research documents and does not include executable code. However, if you discover a security concern (e.g., exposed credentials, sensitive data in committed files, or CI/CD misconfiguration), please report it responsibly.

**Email**: security@qontos.io

Please do not open a public issue for security concerns. We will acknowledge receipt within 48 hours and provide a resolution timeline.

## Scope

- Repository configuration and CI/CD workflows
- Accidentally committed secrets or credentials
- Supply-chain concerns in GitHub Actions dependencies
