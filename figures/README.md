# Figures

This directory contains architecture diagrams, system charts, and publication-ready figures used across QONTOS research papers and the whitepaper.

## Index

Figures will be added as papers are published. Expected categories:

| Category | Description |
|----------|-------------|
| `architecture/` | System-level architecture diagrams |
| `hardware/` | Qubit and module layout diagrams |
| `interconnects/` | Photonic interconnect schematics |
| `software/` | Software stack and data flow diagrams |
| `roadmap/` | Roadmap timeline and milestone charts |
| `benchmarks/` | Performance charts and comparison plots |

## Format

- Vector formats (SVG, PDF) preferred for diagrams
- PNG at 300 DPI minimum for raster images
- All figures should include descriptive filenames and alt text in papers

## Usage

All figures in this repository are licensed under Apache 2.0. Attribution is appreciated when reusing figures in external publications.
