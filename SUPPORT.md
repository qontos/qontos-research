# Support

## Getting Help

- **Questions about the research**: Open a [GitHub Issue](https://github.com/qontos/qontos-research/issues) with the `question` label
- **Bug reports**: If you find errors in papers or documents, open an issue with the `bug` label
- **General inquiries**: Contact us at research@qontos.com
