# Contributing to QONTOS Research

Thank you for your interest in contributing to QONTOS Research.

## How to Contribute

This repository contains public research documents, whitepapers, and roadmap materials. Contributions are welcome in the following forms:

- **Corrections**: Typos, factual errors, or broken links
- **Clarifications**: Improvements to explanations or diagrams
- **Figures**: Higher-quality or alternative visualizations
- **Discussion**: Open an issue to discuss research directions or ask questions

## Process

1. Fork this repository
2. Create a feature branch (`git checkout -b fix/typo-in-paper-3`)
3. Make your changes
4. Submit a pull request with a clear description of the change

## Guidelines

- Keep pull requests focused on a single change
- Use clear, descriptive commit messages
- Ensure all markdown links are valid
- Do not commit binary files (PDFs, images) without prior discussion

## Code of Conduct

Be respectful and constructive in all interactions. We are building the future of quantum computing together.
